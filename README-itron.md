# Balloon Popping Challenge

## Installation

```sh
# Clone the repository
git clone https://github.com/NYCU-iTron/BalloonPoppingChallenge-2026
cd BalloonPoppingChallenge-2026

# Clone the submodules
git submodule update --init

# Setup the Python environment
uv sync
```

## Useful Commands

To add the official repository as an upstream remote:

```sh
git remote add upstream https://github.com/ARRC-Rocket/BalloonPoppingChallenge
```

To pull the latest changes from the official repository:

```sh
git checkout main
git pull upstream main
```
